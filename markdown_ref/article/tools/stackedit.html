StackEdit

主页: StackEdit
平台: Web
费用: 免费、开源
技术: Web

介绍

StackEdit 是一款基于 Web 技术开发的 Markdown 编辑器，它可以将编辑的文档发布到 Blogger、Wordpress，也可以与 Dropbox、Github、Google Drive 同步。
