Numbers

在Python中一共有三种数字类型:

  • int 整数型 (e.g. 2, 4, 20)
    • bool 布尔 (e.g. False and True, acting like 0 and 1)
  • float 浮点型(e.g. 5.0, 1.6)
  • complex 复数型(e.g. 5+6j, 4-3j)
